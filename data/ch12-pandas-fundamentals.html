Chapter 12:
Pandas Fundamentals

NumPy handles numbers in arrays. Pandas handles TABLES — rows and columns, with labels, mixed types, and missing values — which is what almost every real dataset actually looks like. This is the tool you'll use to load, inspect, filter, and summarize data for the rest of this course.

🕐 ~55 min read 💻 5 exercises 🏗️ 1 mini project 🧪 Quiz to complete
12.1 — DataFrame: The Spreadsheet Inside Python

A DataFrame is a table: rows and named columns, like a spreadsheet. A Series is a single column (or row) of it. Remember Chapter 4's "list of dicts" pattern? A DataFrame is that same idea, but built for fast, convenient bulk operations.

Creating a DataFrame from a dict of lists
import pandas as pd data = { 'name': ['Alice', 'Bob', 'Carol'], 'score': [92, 75, 81], 'city': ['Hubli', 'Pune', 'Mumbai'], } df = pd.DataFrame(data) print(df) # pandas prints a nicely formatted table print(f'Shape: {df.shape}') # (3, 3) — 3 rows, 3 columns print(f'Columns: {list(df.columns)}')
12.2 — Reading a Real CSV File

Chapter 5 parsed CSVs manually with the csv module. Pandas does the whole thing — including type conversion — in one line: pd.read_csv().

One line replaces a whole csv.DictReader loop
import pandas as pd csv_text = 'name,age,score\nAlice,28,92\nBob,34,75\nCarol,25,81\n' with open('people.csv', 'w') as f: f.write(csv_text) df = pd.read_csv('people.csv') # numbers are ALREADY int/float — no manual conversion needed print(df.dtypes) # shows the type pandas inferred for each column print(f'Average age: {df["age"].mean():.1f}')
12.3 — Indexing: .loc vs .iloc

.iloc selects by integer POSITION (like a list index). .loc selects by LABEL (the actual index value, or column name) — for a default DataFrame these often look the same, but they mean different things and diverge the moment your index isn't a simple 0,1,2....

.loc and .iloc side by side
import pandas as pd df = pd.DataFrame({'name': ['Alice','Bob','Carol'], 'score': [92,75,81]}) print(df.iloc[1]) # the 2nd ROW by POSITION — Bob's row print(df.loc[1, 'name']) # the value at index label 1, column 'name' — 'Bob' print(df['score']) # a whole column, as a Series print(df.head(2)) # first 2 rows
12.4 — Filtering Rows

Same boolean-masking idea from Chapter 11, applied to a DataFrame: df[condition] keeps only the matching rows. Combine conditions the same way — & / |, each side in parentheses.

Filtering a DataFrame
import pandas as pd df = pd.DataFrame({ 'name': ['Alice','Bob','Carol','Dave'], 'score': [92,75,81,60], 'dept': ['Sales','Eng','Sales','Eng'], }) qualified = df[df['score'] >= 80] print(f'Qualified: {list(qualified["name"])}') sales_high = df[(df['dept'] == 'Sales') & (df['score'] >= 80)] print(f'Sales & qualified: {list(sales_high["name"])}')
12.5 — Sorting

.sort_values('column') sorts a DataFrame by any column, ascending by default.

Ranking by score
import pandas as pd df = pd.DataFrame({'name': ['Alice','Bob','Carol'], 'score': [92,75,81]}) ranked = df.sort_values('score', ascending=False) print(f'Ranked: {list(ranked["name"])}') # highest score first
12.6 — GroupBy: Aggregating by Category

.groupby('column') splits a DataFrame into groups sharing a value, so you can aggregate each group separately — "average score PER department," not just overall.

Average score by department
import pandas as pd df = pd.DataFrame({ 'name': ['Alice','Bob','Carol','Dave','Eve'], 'dept': ['Sales','Eng','Sales','Eng','Sales'], 'score': [85,72,91,68,77], }) avg_by_dept = df.groupby('dept')['score'].mean() print(avg_by_dept) print(f'Sales average: {avg_by_dept["Sales"]:.4f}') # value_counts() — how many rows per category print(df['dept'].value_counts())
⭐
groupby is the pandas analogue of NumPy's axis=0 aggregation — instead of collapsing by position (rows/columns), it collapses by CATEGORY. Same mental move as Chapter 11, applied to labeled data.
Chapter 12 Summary
  • DataFrame = a table (rows + named columns). Series = a single column.
  • pd.read_csv() loads and type-converts a CSV in one line.
  • .loc selects by label; .iloc selects by position.
  • df[condition] filters rows — same boolean masking as NumPy, combined with & / |.
  • .sort_values() ranks by any column. .groupby().agg() summarizes by category.
