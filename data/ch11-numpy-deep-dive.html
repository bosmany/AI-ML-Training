Chapter 11:
NumPy Deep Dive

Welcome to Module 3 — Data Tools. Chapter 8 introduced vectors, matrices, and the dot product. This chapter turns NumPy from "new syntax" into a reflex: broadcasting, boolean masking, and axis-based aggregation are the three techniques that let you replace almost every for-loop over data with a single fast expression.

🕐 ~55 min read 💻 5 exercises 🏗️ 1 mini project 🧪 Quiz to complete
11.1 — Why Vectorized Code Beats Loops

A Python for loop processes one element at a time, with real overhead per iteration. NumPy's vectorized operations push the loop down into pre-compiled, low-level code — the same math, done all at once. The rule from here on: if you're writing a Python loop over a NumPy array to do math, there is almost always a vectorized way to avoid it.

11.2 — Array Creation Shortcuts

Beyond np.array() from a list, NumPy has generators for common patterns: ranges, evenly-spaced points, and pre-filled arrays.

Creating arrays without writing out every value
import numpy as np print(list(np.arange(0, 10, 2))) # [0, 2, 4, 6, 8] — like range(), but returns an array print(list(np.linspace(0, 1, 5))) # [0.0, 0.25, 0.5, 0.75, 1.0] — 5 evenly spaced points from 0 to 1 print(np.zeros((2, 3)).shape) # (2, 3) — a 2x3 grid of zeros, useful as a starting point print(np.ones(4)) # [1. 1. 1. 1.]
11.3 — Broadcasting: Operating on Different Shapes

Broadcasting lets NumPy apply an operation between arrays of different (but compatible) shapes, by stretching the smaller one conceptually. You already used the simplest form in Chapter 8 (array + scalar). Here's the powerful version: matrix ± a single row.

Mean-centering a matrix column-wise — no loop over columns
import numpy as np matrix = np.array([[1,2],[3,4],[5,6]]) # shape (3, 2) col_means = matrix.mean(axis=0) # shape (2,) — one mean per column print(f'Column means: {list(col_means)}') # [3.0, 4.0] centered = matrix - col_means # (3,2) - (2,) -> broadcasts: subtracted from EVERY row print(f'Centered:') print(centered)
⭐
The broadcasting rule in one sentence: shapes are compatible if, comparing from the right, each dimension either matches or one of them is 1 (or missing). (3,2) and (2,) match because 2=2, and the missing dimension is treated as "stretch to fit."
11.4 — Boolean Masking: Filtering Without a Loop

Comparing an array to a value gives you an array of True/False. Using THAT as an index filters the array — no loop, no if statement per element. Combine conditions with & (and) / | (or) — NOT Python's and/or, which don't work elementwise. Always wrap each condition in parentheses.

Filtering with combined conditions
import numpy as np scores = np.array([45, 62, 78, 55, 90, 33, 71, 88]) mask = scores >= 60 print(f'Mask: {list(mask)}') # [False, True, True, False, True, False, True, True] print(f'Passing: {list(scores[mask])}') # [62, 78, 90, 71, 88] — only the True positions # Combine conditions: passing, but not top-tier combined = scores[(scores >= 60) & (scores <= 85)] print(f'Mid-range: {list(combined)}')
11.5 — Aggregating Along an Axis

For a 2D array, axis=0 aggregates DOWN each column (collapsing rows); axis=1 aggregates ACROSS each row (collapsing columns). This trips up almost everyone at first — the axis you name is the one that DISAPPEARS from the result.

Row sums vs column means — a grade book
import numpy as np # 3 students (rows) x 3 subjects (columns) grades = np.array([ [80, 90, 70], [60, 85, 95], [75, 65, 88], ]) row_sums = grades.sum(axis=1) # collapse columns -> total per STUDENT col_means = grades.mean(axis=0) # collapse rows -> average per SUBJECT print(f'Total per student: {list(row_sums)}') print(f'Average per subject: {[round(float(v),2) for v in col_means]}')
11.6 — Reshaping Arrays

.reshape() changes an array's shape without changing its data or order — useful whenever a library expects a specific shape (like a flat list of pixels reshaped into an image grid).

Reshape and flatten
import numpy as np flat = np.arange(12) # [0, 1, 2, ..., 11] — shape (12,) grid = flat.reshape(3, 4) # reinterpret as 3 rows x 4 columns print(f'Original shape: {flat.shape}, reshaped: {grid.shape}') print(grid) back_to_flat = grid.flatten() # undo it — back to shape (12,) print(f'Flattened again: {list(back_to_flat)}')
11.7 — Putting It Together: Vectorized Standardization

Chapter 9 computed a z-score for a single vector. Now do it for an ENTIRE feature matrix at once — every column standardized in one broadcasting expression, no loop over features.

Standardizing every column of a dataset, in one line each
import numpy as np X = np.array([ [10, 1], [20, 2], [30, 3], [40, 4], ], dtype=float) mean = X.mean(axis=0) # one mean PER COLUMN — shape (2,) std = X.std(axis=0) # one std PER COLUMN — shape (2,) standardized = (X - mean) / std # broadcasts across every row — the whole dataset, one line print(standardized)
💡
This is exactly what StandardScaler does in scikit-learn (Module 4). Every feature gets rescaled to mean 0, std 1 — so a model doesn't accidentally treat "salary in dollars" as more important than "years of experience" just because its raw numbers are bigger.
Chapter 11 Summary
  • Vectorized operations replace loops — faster, and usually more readable once you're fluent in the pattern.
  • np.arange, np.linspace, np.zeros/ones generate arrays without writing every value.
  • Broadcasting lets a smaller array (like a row of column-means) apply across a whole matrix.
  • Boolean masking (arr[arr > x]) filters without a loop. Combine conditions with & / |, each side in parentheses.
  • axis=0 collapses rows (down each column); axis=1 collapses columns (across each row).
  • .reshape() / .flatten() change an array's shape without touching its data.
